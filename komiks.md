# KOMIKS: "The Power User Awakening"

**Styl:** XKCD/CommitStrip - prosty line art, czarno-biały, minimalizm
**Panele:** 4
**Bohaterowie:** Dev (ewolucja przez czas), Boss (panel 4)

---

### PANEL 1: Week 1 - Blind Trust

**KOMPOZYCJA:** Plan średni, z boku

**BOHATEROWIE:**
- Dev (~28 lat): Siedzi przy biurku, prosty uśmiech naiwnego optymizmu, potargane włosy, t-shirt z logo GitHub. Patrzy na laptop.

**PRZEDMIOTY/TŁO:**
- Laptop na biurku (ekran do widza), kubek kawy (lewy róg), myszka
- Minimalistyczne biurko, puste ściany

**TEKST:**
- Narracja: "Week 1"
- Dev (głośno, do laptopa): "Read the file!"
- Terminal na ekranie:
  ```
  > Claude, read logs/error.log
  ✓ Reading... [entire file]
  ```
- Dev (myśl): "Magic!"
  - Dymek: chmurka, nad głową

**DETALE:** Naiwny wyraz twarzy, prostota

---

### PANEL 2: Week 2 - The Doubt

**KOMPOZYCJA:** Zbliżenie na twarz + ekran, perspektywa lekko z góry

**BOHATEROWIE:**
- Dev: Lekko zmarszczone brwi, palec na brodzie (póz myśliciela), pochylony nad ekranem

**PRZEDMIOTY/TŁO:**
- Ekran laptopa (większy niż w Panel 1), kubek kawy (teraz pusty, przewrócony)

**TEKST:**
- Narracja: "Week 2"
- Ekran:
  ```
  > Find the API function
  Searching... found 3 files
  (sometimes 12, sometimes 0...)
  ```
- Dev (dialog, frustracja): "Wait... why is it different every time?"
  - Dymek: normalny, z prawej

**DETALE:** Token "?" nad głową (mała ikonka)

---

### PANEL 3: Month 1 - The Discovery

**KOMPOZYCJA:** Plan średni, dev lekko wyprostowany (postawa pewności)

**BOHATEROWIE:**
- Dev: Szeroki uśmiech (eureka!), oczy błyszczą, wyprostowana postawa. Ręce nad klawiaturą (póz hakera z filmu).

**PRZEDMIOTY/TŁO:**
- Laptop, kubek pełen kawy (świeża!), kartka z notatkami (leży obok)
- Ekran większy, więcej szczegółów

**TEKST:**
- Narracja: "Month 1 - The Awakening"
- Ekran:
  ```
  > Use Grep with:
    - multiline: true
    - pattern: 'api\..*'
    - output_mode: "content"

  ✓ Found 47 matches (exact!)
  ```
- Dev (wykrzyknik): "I can CONTROL this!"
  - Dymek: z promieniami (eksplozja radości)

**DETALE:** Kartka z napisem "Tool Params Cheat Sheet"

---

### PANEL 4: Month 2 - Full Power + Punchline

**KOMPOZYCJA:** Szersza scena, dev z lewej, boss z prawej (stoi), między nimi biurko

**BOHATEROWIE:**
- Dev: Spokojny, pewny siebie, lekki uśmiech, wyprostowany. Kubek kawy z napisem "I AM THE CAPTAIN NOW". Okulary (nowa wersja - profesjonalista).
- Boss (~45 lat): Stoi, lekko zdziwiony, trzyma tablet, biznesowa koszula

**PRZEDMIOTY/TŁO:**
- Laptop, kubek ("CAPTAIN"), kartki z notatkami (stos), drugi monitor (w tle)
- Boss trzyma tablet z napisem "Task: DONE (5 min)"

**TEKST:**
- Narracja: "Month 2"
- Ekran (wieloliniowy kod):
  ```
  1. Grep offset:0 limit:1000
  2. Read file_path:X offset:50k
  3. WebFetch url:Y prompt:"..."
  4. Write → report.md
  ```
- Boss (zdziwienie): "You finished the analysis in 5 minutes?!"
  - Dymek: normalny, nad nim
- Dev (spokój, pewność): "I learned to DRIVE the tools, not just ride them."
  - Dymek: normalny, spokojny ton
- Dev (myśl, mały tekst): "(offset, limit, multiline... my precious...)"
  - Dymek: mała chmurka, kursywą

**DETALE:**
- Na monitorze widać "Task completed: 4/4"
- Boss lekko otwarte usta (zaskoczenie)
- Dev ma postawę absolutnego spokoju (zen master)

---

**KONIEC**

**Humor:** Ewolucja od naiwności → frustracji → eureki → zen mastery. Relatable dla każdego kto przeszedł tę drogę z AI tools.

**Punchline:** "DRIVE not ride" + tolkienowskie "my precious" o parametrach = geek culture + practical wisdom
