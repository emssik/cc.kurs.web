# The Renovation Crew

**Styl:** XKCD/CommitStrip - czarno-biały line art
**Liczba scen:** 5
**Bohaterowie:** Junior dev - młody programista w za dużej bluzie z kapturem, entuzjastyczny; Boss - manager w koszuli, trzyma tablet z wykresami; Senior dev - brodaty, spokojny, kubek kawy; 5 agentów-robotów - proste prostokąty z oczami, oznaczone #1-#5

---

### PANEL 1: Wielkie odkrycie

**KOMPOZYCJA:** Plan średni, od frontu

**BOHATEROWIE:**
- Junior dev: Siedzi przy biurku, oczy rozszerzone, ręce nad klawiaturą, ekstatyczny uśmiech. Bluza z kapturem, potargane włosy.

**PRZEDMIOTY/TŁO:**
- Monitor z terminalem, biurko, kubek z kawą, naklejka "I ♥ AI" na laptopie

**TEKST:**
- Narracja: "Monday. Junior discovers Agent Teams."
- Ekran: `typo in login.ts line 42: "pasword" → "password"`
- Dialog Junior (myśl): "One agent? No. This calls for a TEAM."
  - Dymek: chmurka myśli, nad głową

**DETALE:** Post-it na monitorze: "fix typo — ETA: 2 min"

---

### PANEL 2: Mobilizacja

**KOMPOZYCJA:** Plan szeroki, lekko z góry (bird's eye)

**BOHATEROWIE:**
- Junior dev: Stoi z rozłożonymi rękami jak dyrygent, dumna poza, uśmiech pewności.
- 5 agentów-robotów: Ustawieni w półkolu przed nim. Każdy ma etykietę: #1 SECURITY, #2 PERFORMANCE, #3 TESTS, #4 DOCS, #5 ARCHITECT.

**PRZEDMIOTY/TŁO:**
- Minimalistyczne biuro, w tle tablice z diagramami

**TEKST:**
- Dialog Junior: "Team, we have a CRITICAL mission. Agent #1 — audit the security implications of this typo. #5 — design the architecture for the fix."
  - Dymek: normalny, duży, nad sceną
- Dialog Agent #3 (mały dymek): "...it's one character."
  - Dymek: normalny, mały, z boku

**DETALE:** Agent #5 ma narysowany mały znak zapytania nad głową

---

### PANEL 3: Chaos operacyjny

**KOMPOZYCJA:** Plan szeroki, przekrój przez 5 mini-paneli w jednym (komikowy split screen)

**BOHATEROWIE:**
- Każdy agent w swoim mini-panelu, zajęty pracą

**PRZEDMIOTY/TŁO:**
- 5 mini-ekranów z terminalami

**TEKST:**
- Agent #1: "Loading CLAUDE.md... Loading MCP... Loading skills..."
- Agent #2: "Running benchmarks on string 'pasword' vs 'password'..."
- Agent #3: "Writing 47 test cases for one-char change..."
- Agent #4: "Generating RFC: 'Password Field Naming Convention v2.1'..."
- Agent #5: "Drafting migration plan across 0 affected services..."
- Narracja (na dole): "45 minutes later. Five context windows. Five full sessions."

**DETALE:** Mały licznik tokenów w rogu: "Tokens: 847,293 and counting"

---

### PANEL 4: Rachunek

**KOMPOZYCJA:** Plan średni, z boku

**BOHATEROWIE:**
- Boss: Stoi z tabletem, brwi uniesione, usta otwarte. Koszula, poważna mina.
- Junior dev: Skulony przy biurku, nerwowy uśmiech, pot na czole.

**PRZEDMIOTY/TŁO:**
- Tablet Bossa pokazuje wykres kosztów (linia gwałtownie w górę), biurko Juniora

**TEKST:**
- Dialog Boss: "Why does today's API bill look like a hockey stick?"
  - Dymek: normalny, z lewej
- Dialog Junior: "We used cutting-edge multi-agent orchestration!"
  - Dymek: normalny, z prawej
- Dialog Boss: "For WHAT?"
  - Dymek: krzyk (ząbkowane krawędzie), z lewej

**DETALE:** Na tablecie widoczna kwota: "$$$"

---

### PANEL 5: Puenta

**KOMPOZYCJA:** Plan średni, od frontu, podzielony na dwie części (przed/po)

**BOHATEROWIE:**
- Lewa strona (BEFORE): Junior z 5 agentami, chaos, strzałki, diagramy, mnóstwo dymków
- Prawa strona (AFTER): Senior dev — brodaty, spokojny, kubek kawy — siedzi przy biurku, jeden palec na klawiaturze

**TEKST:**
- Nad lewą stroną: "The Renovation Crew approach"
- Pod lewą: "5 agents, 45 min, 847K tokens"
- Nad prawą stroną: "The Senior approach"
- Ekran Seniora: `s/pasword/password/`
- Pod prawą: "0 agents, 2 seconds, 0 tokens"
- Narracja (na dole, kursywą): "You don't hire a renovation crew to hang a picture."

**DETALE:** Kubek Seniora: "Works on my machine"
