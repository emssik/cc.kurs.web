# The Microphone Incident

**Styl:** XKCD/CommitStrip - czarno-bialy line art
**Liczba scen:** 5
**Bohaterowie:** Junior dev - mlody programista w koszulce z logo mikrofonu, sluchawki na szyi; Senior dev - brodaty mentor z kubkiem kawy, siedzi przy sasiednim biurku; Security Officer - kobieta w garniturze z identyfikatorem "SEC", surowa mina

---

### PANEL 1: Discovery

**KOMPOZYCJA:** Plan sredni, biurko Juniora z boku

**BOHATEROWIE:**
- Junior: Siedzi przy biurku, oczy szeroko otwarte, usmiech odkrywcy. Jedna reka na klawiaturze, druga wskazuje na monitor.

**PRZEDMIOTY/TLO:**
- Open space - rzedy biurek w tle (zarysowane minimalistycznie). Monitor z napisem. Kubek z kawą.

**TEKST:**
- Narracja: "9:02 AM. Open space. 47 people within earshot."
- Ekran monitora: "Voice Mode: ON | Push-to-talk: SPACE"
- Dialog Junior: "Oh. My. God. I can TALK to it?"
  - Dymek: normalny, pozycja: nad glowa

**DETALE:** Na monitorze maly icon mikrofonu

---

### PANEL 2: First taste

**KOMPOZYCJA:** Plan sredni-bliski, Junior mowi do laptopa, Senior w tle zerka znad kubka

**BOHATEROWIE:**
- Junior: Pochylony do laptopa, trzyma SPACE, mowi pewnie. Lekki usmieszek.
- Senior: W tle, kubek przy ustach, zerka jednym okiem. Spokoj.

**PRZEDMIOTY/TLO:**
- Fale dzwiekowe wychodzace z ust Juniora (linie). Biurka w tle.

**TEKST:**
- Junior (glosno): "Add input validation to the login form in auth.ts!"
  - Dymek: normalny z pogrubionym tekstem (glosno), pozycja: nad glowa
- Senior (mysl): "Hm. Not bad."
  - Dymek: chmurka mysli, pozycja: nad glowa

**DETALE:** Osoby w tle lekko odwracaja glowy

---

### PANEL 3: Escalation

**KOMPOZYCJA:** Plan szerszy, Junior stoi teraz, gestykuluje, caly open space slucha

**BOHATEROWIE:**
- Junior: Stoi przy biurku, rece rozlozone, mowi glosno i entuzjastycznie. Oczy zamkniete (w transie brainstormu).
- Pracownicy w tle: 3-4 stick figures, odwrocone w strone Juniora, znaki zapytania/wykrzykniki nad glowami.

**PRZEDMIOTY/TLO:**
- Duze fale dzwiekowe. Biurka, monitory.

**TEKST:**
- Junior (glosno): "...and the staging database password is 'admin123', which we should REALLY change, and the AWS keys are in that .env file on John's desktop..."
  - Dymek: duzy, normalny, pozycja: nad glowa, tekst mniejsza czcionka zeby sie zmiescil
- Narracja: "Minute 7. The brainstorm mode has taken over."

**DETALE:** W tle ktos notuje na kartce. Inna osoba robi zdjecie telefonem.

---

### PANEL 4: The arrival

**KOMPOZYCJA:** Plan sredni, Security Officer stoi za Juniorem. Junior jeszcze nie widzi.

**BOHATEROWIE:**
- Junior: Nadal mowi z zamknietymi oczami, gestykuluje.
- Security Officer: Stoi za nim, rece skrzyzowane, jedna brew uniesiona. Identyfikator "SEC" widoczny.
- Senior: W tle, zakrywa twarz dlonia (facepalm).

**PRZEDMIOTY/TLO:**
- Security Officer rzuca cien na biurko Juniora.

**TEKST:**
- Junior (glosno): "...and the production API keys rotate every— "
  - Dymek: normalny, urwany (z "—")
- Security Officer: *tap tap* (na ramieniu Juniora)

**DETALE:** Na monitorze Juniora Claude odpowiedzial: "I strongly recommend not saying credentials out loud."

---

### PANEL 5: The punchline

**KOMPOZYCJA:** Plan sredni-bliski, Junior odwrocony, blady. Security Officer z notatnikiem.

**BOHATEROWIE:**
- Junior: Odwrocony na krzesle, blady, pot na czole, usta otwarte. Rece przy klawiaturze (puścil SPACE).
- Security Officer: Trzyma notatnik, lekki usmiech (ironiczny). Pisze.

**PRZEDMIOTY/TLO:**
- Monitor w tle z Claude odpowiedzia.

**TEKST:**
- Security Officer: "Thanks for the security audit. The whole floor took notes. You're saving me weeks of work."
  - Dymek: normalny, pozycja: z prawej
- Junior: "I was just... using Voice Mode..."
  - Dymek: normalny (maly, cichy), pozycja: z lewej
- Narracja: "Pro tip: Voice Mode + sensitive data + open space = involuntary penetration test."

**DETALE:** Na notatniku Security Officer widac liste: "admin123 ✓, AWS keys ✓, .env location ✓"
